# Instalar paquetes necesarios para WordProfe

## Step 1 – Install Java 8
```
sudo apt update
sudo apt install openjdk-8-jdk openjdk-8-jre openjfx

```

## Step 2 – Verify Java Installation

```
java -version

```

## Step 3 – Setup JAVA_HOME and JRE_HOME Variable
```
cat >> /etc/environment <<EOL
JAVA_HOME=/usr/lib/jvm/java-8-openjdk-amd64
JRE_HOME=/usr/lib/jvm/java-8-openjdk-amd64/jre
EOL
```

## Step 4 - Downloading Tomcat Archive
```
wget https://downloads.apache.org/tomcat/tomcat-8/v8.5.59/bin/apache-tomcat-8.5.59.tar.gz

tar xzf apache-tomcat-8.5.59.tar.gz

mv apache-tomcat-8.5.59 /usr/local/tomcat8

```
## Step 5 – Configure Environment Variables
```
echo "export CATALINA_HOME="/usr/local/tomcat8"" >> ~/.bashrc

source ~/.bashrc

```
## Step 6 – Start Tomcat Server
```
cd /usr/local/tomcat8

./bin/startup.sh
```
### Config Tomcat Server
```
nano /usr/local/tomcat8/conf/server.xml
```
